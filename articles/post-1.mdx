# Ini adalah Judul Berkas MDX

Ini adalah contoh berkas MDX. Anda dapat mencampurkan teks Markdown dengan kode JSX dan bahkan menambahkan metadata di atas kontennya.

```jsx
import React from "react";

const MyComponent = () => {
  return <div classname="text-lg">Selamat datang ke konten MDX ini!</div>;
};

<MyComponent />;
```
